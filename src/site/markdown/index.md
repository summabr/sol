# Overview

The [Summa] Open Library is a general-purpose set of open source Java components.

[Summa]: http://summa.com.br/


## Usage


```xml
<project>
  ...
  <dependencies>
    <dependency>
      <groupId>br.com.summa</groupId>
      <artifactId>sol</artifactId>
      <version>0.11-SNAPSHOT</version>
      <scope>run</scope>
    </dependency>
  </dependencies>
  ...
</project>
```
